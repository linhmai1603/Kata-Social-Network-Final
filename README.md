This project is built to mimic Twitter with three main fuctions where the user can publish a message, follow other people and check message on their wall.


Instruction:
Requirement: Java 1.8+
	     Maven Apache (4.0.0)

TO Build the project: from the console using "mvn clean install"

To run the Application: Right click on App.java in src/main/java/kata_socialnetwork and choose run as Java Application
